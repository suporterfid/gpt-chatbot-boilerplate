Integration Examples

1. Floating Chat Widget (Bottom Right)

Perfect for customer support and general assistance

<script src="app.js"></script>
<script>
ChatBot.init({
mode: 'floating',
position: 'bottom-right',
title: 'Chat with us',
primaryColor: '#1FB8CD'
});
</script>

2. Inline Chat Widget

Embed directly into your page content

<div id="inline-chatbot"></div>
<script>
ChatBot.init('#inline-chatbot', {
mode: 'inline',
height: '500px',
welcomeMessage: 'How can I help you today?'
});
</script>

3. Custom Themed Widget

Fully customized to match your brand

ChatBot.init('#custom-chatbot', {
theme: {
primaryColor: '#8B5CF6',
backgroundColor: '#F3F4F6',
fontFamily: 'Inter, sans-serif',
borderRadius: '12px'
},
assistant: {
name: 'AI Assistant',
avatar: '/assets/bot-avatar.png'
}
});

Configuration Options

Basic Options

  • mode - 'floating' or 'inline'
  • position - 'bottom-right', 'bottom-left'
  • title - Chat widget title
  • height - Widget height (inline mode)
  • width - Widget width

Styling

  • primaryColor - Main theme color
  • backgroundColor - Background color
  • fontFamily - Custom font family
  • borderRadius - Corner rounding
  • shadow - Drop shadow

Assistant Settings

  • name - Assistant display name
  • avatar - Assistant avatar image
  • welcomeMessage - Initial greeting
  • placeholder - Input placeholder text

Technical

  • apiEndpoint - Backend API URL
  • streamingMode - 'sse', 'websocket', 'auto'
  • maxMessages - Conversation history limit
  • enableMarkdown - Markdown support

Quick Setup

1

Clone Repository

2

Configure Environment

cp .env.example .env
echo "OPENAI_API_KEY=your-api-key-here" >> .env
3

Start with Docker

docker-compose up -d
4

Test Integration

Open http://localhost:8080 in your browser
